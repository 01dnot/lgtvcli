# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-18

### Added
- Initial release
- TV discovery via SSDP and mDNS
- Pairing and configuration management
- Power control (on via Wake-on-LAN, off, screen on/off)
- Volume control (up/down/set/mute)
- Audio output selection
- Application management (list/launch/close)
- Input source switching
- TV channel control
- Media playback controls
- Remote control button commands
- Keyboard input support
- Mouse/pointer control
- System information queries
- Toast notifications
- Multi-TV support
- Comprehensive CLI with subcommands

### Notes
- Screenshot and HTML/DOM capture not supported (LG WebOS API limitation)
- Wake-on-LAN requires MAC address configuration
- Requires webOS 2.0 or higher
